Protaller

Protaller es una aplicación diseñada para la gestión integral de talleres mecánicos. Permite a los talleres mantener un registro organizado de clientes y vehículos, generando formularios y facturas de manera eficiente. Además, la app incluye funcionalidades para asignar tareas a los trabajadores, registrar horas de trabajo en los vehículos y facilitar la ubicación de talleres mediante rastreo GPS, brindando a los clientes acceso a un registro detallado de los servicios realizados en sus vehículos. Protaller está desarrollado en Angular y utiliza una base de datos en Spring Boot.

TutorialHub

TutorialHub es una plataforma para la gestión de clases particulares, donde los profesores pueden publicar y organizar clases con ubicación GPS. Los estudiantes pueden explorar clases por categoría y proximidad, enviar solicitudes de inscripción y valorar a los profesores. Los docentes pueden asignar tareas y actividades específicas, que los estudiantes pueden completar y visualizar en su perfil. Esta app, desarrollada en Angular y Spring Boot, organiza toda la información de las clases, facilitando una experiencia de aprendizaje flexible y accesible.

SwimTrack

SwimTrack es una aplicación para monitores de natación, diseñada para gestionar el rendimiento de los nadadores. Los monitores pueden crear y gestionar perfiles de nadadores, registrar resultados de pruebas de competición y hacer un seguimiento de la evolución en el rendimiento de cada deportista. También permite organizar entrenamientos y competiciones, y cuenta con una agenda para planificar eventos. SwimTrack está desarrollada en Ionic y utiliza Spring Boot para su base de datos.

Spotyfy

Spotyfy es una aplicación educativa para practicar tecnologías de reproducción musical y bases de datos. Este proyecto permite la implementación de funcionalidades básicas de streaming de música y el aprendizaje en gestión de datos multimedia. La aplicación fue desarrollada en Java y FXML como parte de una práctica avanzada de desarrollo de software.

ClínicaPro

ClínicaPro es una innovadora aplicación diseñada para optimizar la gestión integral de clínicas de fisioterapia. Esta herramienta permite a las empresas administrar de forma eficiente clientes, empleados y citas médicas, asegurando un flujo de trabajo organizado y sin contratiempos. ClínicaPro también facilita el manejo de datos esenciales para el día a día de la clínica, desde la programación de sesiones hasta el seguimiento de tratamientos, ofreciendo una experiencia fluida tanto para el personal como para los pacientes. Desarrollada con tecnologías modernas, su interfaz intuitiva y robusta garantiza un control administrativo avanzado. El proyecto fue desarrollado en Angular para el frontend y Spring Boot para el backend.

EquipFix

EquipFix es una aplicación desarrollada en React con una base de datos en Spring Boot, diseñada para gestionar el mantenimiento de equipos informáticos, industriales y eléctricos. Permite llevar un registro detallado de los equipos, generar informes y gestionar su ubicación para facilitar las revisiones. Su objetivo es optimizar el seguimiento del mantenimiento en las empresas, asegurando que los trabajadores estén siempre informados sobre el estado y la localización de los equipos.
